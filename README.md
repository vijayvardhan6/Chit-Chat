## Tech Stack

Client: <b>`React JS`</b>

Server: <b>`Node.js`</b>, <b>`Express.js`</b>

Database: <b>`Mongo DB`</b>

<br>

## Features

- <b>`Authentication`</b>
  - Log In
  - Sign Up
- <b>`User Profile`</b>
- <b>`Search Users`</b>
- <b>`Real-time Chatting with Typing Indicator`</b>
- <b>`Chat Notification`</b>

- <b>`One-to-One Chat`</b>

- <b>`Group Chat`</b>
  - Create Group Chat
  - Update Group Name
  - Add / Remove Users from group
